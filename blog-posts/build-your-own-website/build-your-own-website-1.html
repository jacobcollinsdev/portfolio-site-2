How Does It All Work?

Have you ever wondered how webpages are created? How does your browser know what to display to you, the end-user, when you type in the URL of a page you want to visit?

The answer lies in the fact that your browser is interpreting a number of text files that contain all the instructions required to correctly display the website. There are typically 3 main technologies that are used to power the web:

  1. HTML - adds content and structure to your website (i.e. Header, navigation bar, main content, and footer).
  2. CSS - adds styles to your website (i.e. Colours, formatting, fonts, etc).
  3. JavaScript - adds functionality to your website (i.e. User-driven interactive events such as search bars, games and login buttons).

HTML and CSS are known as mark-up languages and stylesheet languages, respectively. You could make a website using just HTML, but without CSS it would be incredibly boring and wouldn't attract new visitors. HTML provides the structure and content for your webpage, whereas CSS applies all the styles you need to make it visually appealing. What is HTML?

HTML stands for "Hypertext Markup Language". As mentioned above, this is the language that's used to add content and structure to your website. HTML consists of elements which are stored inside something called "tags". These tags, denoted by "<" and ">" brackets, provide information about what kind of element it is. Most tags require an opening and closing tag, as you'll see below. For example, a paragraph tag populated with some text looks like this:

<p>This is some paragraph text.</p>

In the above example, the <p> tag tells the browser that this is a paragraph element. Note that the closing tag, </p>, contains a forward slash after the < symbol which denotes that we are closing the p tag. It's important that you include a closing tag after your content, otherwise your browser will interpret the rest of your code as a paragraph element.

As we move through this tutorial, you will see examples of other tags we can use to add content and everything we use will be explained along the way.

What is CSS?

CSS stands for "Cascading Style Sheets" and uses a concept called inheritance that will enable us to apply styles to all elements of a certain type just by defining the style once. For example, if I wanted my all my paragraphs to have a font-size of 20px (the default font is usually 16px in most browsers), I would apply the following code:

p {
  font-size: 20px;
}

Now, whenever I add a new paragraph, it will have a font-size of 20px. The code above is what's known as a CSS rule. It consists of a selector (the element we're selecting, in this case the p element), a property (in this case 'font-size'), and a value (20px).

By applying CSS rules to HTML elements, we can style a webpage and make them look great. As we move through the tutorial we'll discover different properties we can apply, and these will be explained as we move along.

1. Creating the HTML File

First of all, let's create a new folder called "My Website". Within this, we want to create a new text file by right-clicking the empty space, then New, then Text Document. Name this file "index.html".

2. Setting up the HTML

Copy the following text, exactly as written into your HTML file:

Take note of what tags we have:

<!DOCTYPE html>
The !DOCTYPE tag tells the browser what version of HTML we are using. We'll be using HTML5, which in this case is represented just as 'html'. This part is important because as new versions of HTML are released, some tags are no longer accepted, so we have to tell the browser what standard we are coding to. This does not have a closing tag.

<html>
This tag will contain all our other HTML elements.

<head>
This tag is located between the html and the body tag, and will contain metadata. This is essentially data about the HTML document that isn't displayed on the webpage. This will become clearer in a moment when we start adding other tags inside.

<body>
Contains everything that will be displayed on the webpage.

3. The <head> Tag

As mentioned, the <head> tag contains data about our HTML file. One such type of data we want to provide is the character set we're using. Without this, the browser won't be able to display our page correctly. We can also give our webpage a title, which will be displayed in the browser window.

Note: tags do no require a closing tag.

Now save your document, and go the folder containing our index.html file. Right-click your file, then "Open with" and select your preferred web browser.

You'll see that the browser tab at the top contains the text inside our <title> tag.

4. Adding Content

This is where we can start adding our website content. Remember, this goes within the <body> tag.

First of all we want to give our website a main heading. There are up to six heading types we can use for this. These range from <h1> through to <h6>, but it's not recommended to go further than <h4>. The <h1> tag is the highest level heading element and is often used for the main page title. The <h2> tag is primarily used for sub headings, and <h3> for sub headings within <h2> tags.

Think of what you want your main heading to be and insert this within a <h1> tag inside the <body>.

Now save your file again, and refresh the browser you opened the file up in. It should something like this:

Now add a <h2> underneath your <h1> element with 'About Me' written inside. We can also add a sentence to describe ourselves, written inside a <p> tag.

Save the file again, and reload your browser.

5. Adding Images

To add images to our website, we use the <img> tag. The first thing we have to do, however, is make sure the image we want to use is inside our "My Website" folder.

Now the image is inside our "My Website" folder, we can use the <img> tag to reference it. We also need to include two attributes with our <img> tag: the first is the src attribute, which points to where the image file is located, and the second is the alt attribute, which is the text that will render if the image can't be loaded.

It's up to you where you want to insert your image, I'm going to put it just beneath our <h1> tag.

The src attribute contains path to our image file. In this case, the image file exists in the same folder as our index.html file so all we need to type in is the image file name.

The alt attribute is a short text description of what the image is. If the image can't load for some reason, this text will be displayed instead. It's what screen readers use to describe the image for those with visual impairments.

Save your index.html file again and reload the browser.

I've already pre-sized the image I'm using, so don't worry if your image appears huge in the browser - we can fix this with some CSS, which we'll look at in Part 2 of this tutorial.
