CSS Recap

Remember that CSS is a stylesheet language and stands for "Cascading Style Sheets". This essentially means that the styles we apply "cascade" through the entire document and apply to all elements we've selected until they're overridden.

We declare our styles using CSS rules. These consist of a selector, a property, and a value. Let's have a look at the example for Part 1 again:

p {
  font-size: 20px;
}

The above is an example of a CSS rule for styling paragraph elements.

In this case, the selector is the p element.

The property we want to style is the font-size.

The value we want to assign to the font-size is 20px (px is short for "pixels").

Now all our paragraph elements will have a font-size of 20px. There's so many properties that we can style with CSS and in this tutorial we are going to touch on the basics. If you're hungry for more, I've linked a free online curriculum at the end for you to get your teeth into.

Now we've recapped CSS, let's get started styling the webpage we created in Part 1!

2. Styling the <body>

To get started, let's give our page background and text a different colour. To do this, we'll be using the background-color and color property (note the American English spelling).

Remember how to create a style declaration: we need a selector, a property, and a value.

In this case, we want to select the <body> element, and we want to style two different properties. For the background-color property, I'm going to use something called hex code which is one of a number of different colour coding formats. I won't go into detail on what is this and how it works here, but those of you hungry to learn can consider it as extra curricular work to find out. For now, you just need to know that using hex code allows us to use millions of colours as opposed to just a handful. To assign a colour to our text, I can just write white as the value for the color property, as we don't need to use hex code for simple colours.

You can use this online hex colour picker to help you choose the background colour you want for your site. Copy the code it gives you and use this as the value for background-color.

Save the file and open it in your browser.

If your browser doesn't show the colours you've entered, double check the spelling. Remember to use "color" and not "colour".

We can also use the text-align property to centre our page elements. Again, note the American English spelling "center".

Once again, save and reload in your browser.

3. Changing Text Font and Size

In order to use more exciting fonts, we first have to import them from an external library. I'm going to be using Google Fonts for this. Below are the instructions for importing the font style you want:

Step 1

Head over to Google Fonts and find the fonts you like. I recommend choosing one font for your headings and another for your paragraph text. Select the font you want to use for the headings, and you'll be taken to the information page for that font.

Step 2

Click "Select this style".

Go back to the Google Fonts homepage and select another font for your paragraph text in the same way you've just done.

Step 3

You should see your selected fonts appear on the right hand side of your browser window. Click "Embed" and then within this select "@import".

We want to copy the code between the <style> tags and insert it at the top of our style declarations.

Now we can access these cool new fonts!

To do this we use the font-family property, and pass the name of our font inside single quotation marks as the value. It's also good to add a back-up font-family in case the browser can't render your chosen font. This should be a simple font such as Arial, Helvetica, or Times New Roman to be sure that the browser can access it. Separate your first choice font from your back-up with a comma.

I'm going to give all my h1 and h2 elements the 'Bangers' font, and all my p elements the 'Roboto' font. For both I'll give a back-up font of Arial. Since we're giving the h1 and h2 elements the same font-family, we don't need to create two separate CSS rules - we can simply write h1, h2 as our selectors.

Save and reload your browser.

To change your size of your text, use the font-size property. Most browsers use a default of 16px. Play around with different values for your h1, h2, and p elements.

When the browser reads the first rule, it will apply a font size of 40px to all h1 and h2 elements. When it reads the newly declared h2 rule underneath, it will override the previously declared font size of 40px and apply the new value of 32px. Finally, I added another rule to set a font size of 20px for our p elements.

4. Styling the Image

There are a number of things we can do with our image. We can change its size using the height and width properties. We can also round the corners of the image to make it less square.

Select the image tag using img, then set a value for it's height (I'm going to set a height of 500px but you can experiment around with higher and lower values).

Save and reload your browser. Notice that the browser automatically scaled up the width of the image even though we only set the height property.

Now to change the rounding of the corners of the image, we're going to use the border-radius property and set a value in either pixels or percentages. The higher the value, the more rounded the corners will be. If you set a value as a percentage, the maximum you can use is 50%. There is a difference in how the code is interpreted depending on whether you use pixels or percentages. Have a go at both and see if you can pin-point what it is.

Save the file and reload your browser.
