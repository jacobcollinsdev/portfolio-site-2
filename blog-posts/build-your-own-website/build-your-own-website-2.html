CSS Recap

Remember that CSS is a stylesheet lanuage and stands for "Cascading Style Sheets". This essentially means that the styles we apply "cascade" through the entire document and apply to all elements we've selected until they're overridden.

We declare our styles using CSS rules. These consist of a selector, a property, and a value. Let's have a look at the example for Part 1 again:

p {
  font-size: 20px;
}

The above is an example of a CSS rule for styling paragraph elements.

In this case, the selector is the p element.

The property we want to style is the font-size.

The value we want to assign to the font-size is 20px (px is short for "pixels").

Now all our paragraph elements will have a font-size of 20px. There's so many properties that we can style with CSS and in this tutorial we are going to touch on the basics. If you're hungry for more, have a look at the resources listed at the end.
